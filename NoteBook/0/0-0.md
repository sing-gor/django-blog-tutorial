# 开发环境配置
本次教程会使用到下面到。  

## 安装Git
本次教程全程使用Git进行版本管理。  
不会Git也没有关系，会**复制粘贴**就行。   
### Windows
到[Git的官网](https://git-scm.com/)下载对应操作系统后，一直下一步下一步安装就行了。  
### Unix/Linux
如果您使用的是 Ubuntu ，可以直接使用 apt-get 来安装 Git:  
```bash
sudo apt-get install
```
如果是 CentOS，则是使用 yum 来安装:  
```bash
sudo yum install ruby
```
### Mac
```bash
brew install git
```
